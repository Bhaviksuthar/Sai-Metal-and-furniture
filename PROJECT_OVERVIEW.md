# Sai Metal And Furniture App Overview

This repository will contain the Android application for Sai Metal And Furniture.

Planned core modules:

- Customer-facing gallery of completed metal and furniture work
- Service catalog for fabrication and furniture offerings
- Quote and inquiry submission
- Admin login for owners
- Gallery management for adding and updating completed work
- Client billing and payment tracking
- Project progress management

Suggested technology stack:

- Native Android with Kotlin
- Jetpack Compose UI
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
